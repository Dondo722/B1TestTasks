Данная программа представляет собой веб-интерфейс для:
  1. Загрузки файлов excel формата в СУБД
  2. Просмотра загруженных файлов
  3. Отображение данных из СУБД по визуальной аналогии с excel-файлом для каждого из загруженных файлов
  4. Сохранения файлов с сервера
Программа должна справляться со всеми файлами, оформленными согласно примеру
